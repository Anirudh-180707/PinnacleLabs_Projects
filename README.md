# Pinnacle Labs Internship Projects

This repository contains projects completed as part of the Pinnacle Labs Internship Program. The projects focus on Machine Learning, Natural Language Processing (NLP), and Computer Vision using Python.

## Projects Included

### Task 1 – Fake News Detection

A Machine Learning-based system that classifies news articles as Real or Fake using NLP techniques.

### Task 2 – Road Lane Detection

A Computer Vision project that detects and highlights road lane markings from images or video streams.

### Task 3 – Sentiment Analysis

An NLP project that analyzes textual data and determines the sentiment expressed in customer reviews or comments.

### Task 4 – Chatbot Using Python

A Python-based chatbot designed to provide automated responses and interactive user communication.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* OpenCV
* Natural Language Processing (NLP)
