# 🤖 GUI Chatbot Using Python

## 📌 Overview

This project is a Graphical User Interface (GUI) Chatbot developed using Python and Tkinter. The chatbot is capable of interacting with users through predefined responses and provides a simple conversational experience through an interactive chat window.

The objective of this project is to demonstrate the fundamentals of chatbot development, user interaction, and GUI application design using Python.

---

## 🎯 Features

* Interactive Graphical User Interface (GUI)
* Real-Time User Interaction
* Scrollable Chat Window
* Send Button Functionality
* Enter Key Support
* Predefined Question and Answer System
* User-Friendly Design

## ⚙️ Supported Commands

The chatbot currently supports the following commands and queries:

| User Input               | Chatbot Response                                   |
| ------------------------ | -------------------------------------------------- |
| hello                    | Greets the user                                    |
| hi                       | Displays a welcome message                         |
| how are you              | Responds with a friendly message                   |
| what is python           | Explains Python programming language               |
| what is ai               | Provides information about Artificial Intelligence |
| what is machine learning | Explains Machine Learning                          |
| who created python       | Provides creator information                       |
| bye                      | Displays a goodbye message                         |

### Example Conversation

```text
You: hello
Bot: Hello! How can I help you today?

You: what is python
Bot: Python is a popular programming language used for software development, AI, and data science.

You: what is ai
Bot: AI stands for Artificial Intelligence, which enables machines to mimic human intelligence.

You: bye
Bot: Goodbye! Have a great day.
```

## ▶️ Installation

No external libraries are required.

Tkinter comes pre-installed with most Python distributions.

Verify Python installation:

```bash
python --version
```

---

## ▶️ Running the Project

Execute the following command:

```bash
python main.py
```

A chatbot window will open, allowing users to interact with the system.
